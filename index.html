Le plugin Philips HUE, expose l'API REST des lampes connectées au framework SARAH. Un grand merci à Nicolas Savoini pour son plugin et toutes ses expérimentations.

Getting Started

Indiquer dans les propriétés l'ip du bridge philips (mettre une ip fixe). Le champs alllights représente la liste des toutes les lampes séparée par des vigules.

Lors de la première utilisation il est necessaire de créer un utilisateur. Appuyer sur le bridge, cliquer sur le bouton de la portlet puis raffraichir la fenêtre.

Principes

L'API REST propose de nombreuses options de paramétrage des lampes. L'idée est de ne rien imposer dans le JS pour tout véhiculer depuis le XML. 

<item>
  we are under attack 
  <tag>
    out.action.lights="true"; 
    out.action.rgb="FF0000"; 
    out.action.alert="lselect";
  </tag>
</item>

Pour simplifier le code le plugin gère les couleurs hexadecimales qui sont transformées en HSL et XY.

Le paramètre lights indique de travailler sur toutes les lampes. Sinon il faut préciser light="1" ou group="1". Il est aussi possible de préciser out.action.rgb1="EEDDCC";.
